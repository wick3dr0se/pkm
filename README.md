## Install
```ocaml
$ git clone https://github.com/wick3dr0se/pkm && cd pkm
```

## Usage
```bash
$ pkm
# or if it's installed to path
$ pkm
```

## Controls
H, ←   ...   Back  
J, ↓   ...   Move down  
K, ↑   ...   Move up  
L, →   ...   Do action

Q   ...   Quit
